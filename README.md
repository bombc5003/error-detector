# 딸각에러검출기 (K-SENTINEL Hyper Analyzer)

Python 파일을 드래그 앤 드롭하면 즉시 에러를 검출해주는 Windows 도구입니다.

## 특징
- Python 파일 드롭으로 즉시 에러 검출
- 문법 오류 · 임포트 오류 · 런타임 오류 위치 표시
- 단일 실행 파일 (설치 불필요)
- Windows 10/11 동작
